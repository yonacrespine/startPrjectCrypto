About the Website

This project was carried out by Yona Crespine as part of the John Brice training.
It involves taking data from an API and using it to create a web page.
Here, it concerns coin data where it was necessary to display the name, the symbol, and the different prices depending on the currency.
Subsequently the customer can choose a maximum of 5 corners of their choice. If he chooses more he will have to subtract.

The technologies utilised in this project are:

  • HTML
  • CSS
  • Bootstrap
  • Javascript
